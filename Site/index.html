Silent C2 is a simple, lightweight command-and-control (C2) server designed for use in small-scale projects or as a starting point for larger C2 projects. It's written in Golang & MySQL for its database.

Silent C2 is designed to be easy to set up and use, while providing a good foundation for more complex C2 systems. It currently supports:

  • Command execution
  • API methods
  • Advanced logging
  • And compatibility with Windows, Linux, macOS & Mobile

For more information on how to use Silent C2, check out the documentation.
